## What is this repository?

This repo powers my personal website via **GitHub Pages**. It's a lightweight, static site built with plain **HTML** and **CSS** (no Jekyll or heavy frameworks).

The site serves two main purposes:
1. Professional portfolio showcasing my experience, skills, and support-related mini-projects.
2. Collection of practical walkthroughs and guides, especially around cryptocurrency operations and blockchain observability.
